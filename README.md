## Getting Started

```
go get github.com/blacktop/go-bindiff
```

### `bindiff` demo tool

Install

```
go install github.com/blacktop/go-bindiff/cmd/bindiff@latest
```

Usage

```
bindiff path/to/BinExport1 path/to/BinExport2
```
